# Approximation of Pi using the Monte Carlo method

For n=100,000 randomly placed points.
![results](100000.png)

How the approximation improves with increasing n:

| N         | Approximation |
|   :----:  |   :----:      |
| 1         | 0             |
| 10	      | 2.8000        |
| 100	      | 3.2800        |
| 1000	    | 3.1360        |
| 10000     |	3.1576        |
| 100000	  | 3.1366        |
